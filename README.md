# My Linux dotfiles

## Installations

### Arch

#### First boot

```shell
git clone https://github.com/Harkame/dotfilesLinux.git

cd dotfilesLinux/install/arch

chmod +x ./install.sh

./install.sh
```

#### First log

```shell
git clone https://github.com/Harkame/dotfilesLinux.git

cd dotfilesLinux/install/arch

chmod +x ./end_install.sh

./end_install.sh

chmod +x ./export.sh

./export.sh
```
